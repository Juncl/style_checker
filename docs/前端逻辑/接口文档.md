## 查询用户交付件列表

| 项 | 值 |
|---|---|
| **Mock URL** | `GET /mock/main/rest.root/reviewTool/consistencyCheck/getConsistencyCheckDeliverables` |
| **真实 API** | `GET {ADMIN_BASE_URL}/main/rest.root/reviewTool/consistencyCheck/getConsistencyCheckDeliverables` |
| **前端函数** | `geConsistencyCheckDeliverables()` → `client/src/api/api.ts` |

**出参 `content`**：`ConsistencyCheckDeliverableVo[]`

```ts
interface ConsistencyCheckDeliverableVo {
  id:           number   // 交付件 ID
  name:         string   // 交付件名称
  createTime:   number   // 创建时间（Unix ms）
  creatorId:    number   // 创建人 ID
  filePath:     string   // Pixso 文件路径，如 "290082/290083"
  filePathName: string   // 文件路径可读名称，如 "xxx的项目/xxx的分组"
  platformType: number   // 平台类型（0=hmPhone，1=hmWatch，2=web）
}
```
